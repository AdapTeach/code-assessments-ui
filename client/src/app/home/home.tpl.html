Speed up your AngularJS 1.2 development with a complete and scalable build system that scaffolds the project for you. Just focus on your app, ng-kickstart will take care of the rest.

  Download   Docs on GitHub

AngularJS

The best JavaScript framework out there will power up your awesome app.

Grunt

A smart and scalable grunt based build system will take care of your development workflow, as well as the optimization process for production release. Read more...

Bower

Bower will handle your front-end dependencies.

Sass + Zurb Foundation

Sass is the most mature, stable, and powerful professional grade CSS extension language. Write your CSS in a modular way, the build system will compile your .scss files into a single css files. It should be easy to integrate less, stylus or any other preprocessor if you prefer. Zurb Foundation is the default CSS framework.

API Proxy

If you are developing a single page application tied to a real backend, ng-kickstart will proxy every request to your backend listening on another port. You can configure this of course. Read more...

Modular Structure

Instead of angular-seed monolithic files structure, ng-kickstart comes with a by-feature/files organization, keeping your code organized, especially if you are working on a large code base. If you don't like it, just use your preferred structure, and the build system will still work. Read more...

Keep Your Code Reusable

Every general purpose directive, service or filter, should be placed into a common directory, in this way you can copy and paste the directory into another project, require the module you need, and you are ready to go with your new project.

Unit Testing

The build system comes with a special task for running tests by using Karma Test Runner. Every time you make a production release, unit tests will be run for you.

RESTFul Backend Included

ng-kickstart ships with a working REST api written in 2 versions: One using Silex PHP micro-framework + SQLite and another using expressjs + MongoDB See the getting started section for more information. It would be awesome to have more working examples with other technologies. Please feel free to code, hack and send PR.
