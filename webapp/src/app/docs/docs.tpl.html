What and Why

ng-kickstart is an opinionated kickstart for single page application development in AngularJS. It make you development easy, keeps the structure of the project consistent and allows you to create a fully optimized production release whith a single grunt task. I decided to build this tool because of the lack of a build system that let me develop a single page application keeping an organized file structure, and in the meantime that allows me to develop on a index.html file generated at build time, tied to my real backend.

Getting started

I assume that you have Node.js and SASS installed. You need karma and bower. 


$ sudo npm -g install grunt-cli karma bower

After that install ng-kickstart 


$ git clone git://github.com/vesparny/ng-kickstart yourProjectName
$ cd yourProjectName
$ npm install
$ bower install
$ grunt serve

You are now ready to go, your applcation is avalable at http://127.0.0.1:9000. Every request to /api will be proxied to http://127.0.0.1:9001/api.

In the /backend/silex folder, you can find an example REST api written with Silex PHP micro-framework. Refer to the README.md present in that folder and launch the api. Then go to http://127.0.0.1:9000/notes. Enjoy.
