What is this?

An example of a single page web application (a note app) entirely built on top of angular.js

The backend is written in php with the awesome Silex micro-framework, notes are stored in a local SQLite database.

How do I run it?

From the root of the project run the following commands to install the php and bower dependencies, import some data, and run a local php server (you need at least php 5.4.*).

$ composer installzzz
$ bower install
$ sqlite3 app.db < resources/sql/schema.sql
$ php -S 0:8000 -t web/
